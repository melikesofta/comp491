We use graph layout algorithms to visualize Ekşi Sözlük data by means of entry counts, author counts, and by detecting relationships between topics through the list of common authors between topics.

Our project consists of three major parts of work:

  • crawling through the website and getting entry, topic and author data
  • visualizing the network using graph layout algorithms
  • analyzing the network using social network analysis techniques
